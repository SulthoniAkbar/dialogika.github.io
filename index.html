Dialogika Public Services

Komunitas belajar public speaking yang berfokus kepada pengembangan diri
Berikut adalah program yang kami miliki

Online Class

Mensikapi wabah COVID-19 kami menginisiasi kelas online yang membantu teman-teman diseluruh nusantara untuk menguasai public speaking, di program ini lebih menekankan penguasaan materi. Cocok untuk kamu yang mudah nge-blank, ingin pendadaran, suka disuruh memimpin rapat

Intensive Class

Program intensif adalah program pelatihan public speaking dalam waktu 1 bulan di Yogyakarta, dengan jumlah pertemuan sebanyak 10x. Tidak hanya penguasaan materi, program ini akan membantu teman-teman yang mudah gugup, takut salah, sulit mengembangkan kata dan introvert

MC Class

Program ini akan segera luncur, di program ini akan bertujuan untuk membantu teman-teman didalam menguasai Master of Ceremony. Cocok untuk kamu yang suka diminta menjadi MC ataupun influencer-influencer instagram, youtube ataupun media sosial lainnya
