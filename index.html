Online Class

Public Speaking untuk berbicara secara teratur dan bagaimana cara menyampaikannya secara online adalah Tema untuk belajar Public Speaking via Daring. Berbeda dari beberapa metode pembelajaran daring lainnya. Dialogika menggunakan komunikasi dua arah dan bersifat mini class dengan partisipan yang terbatas. Keunggulan dari program ini adalah partisipan bebas untuk konsultasi dengan mentor

Offline Class

Setelah 3 bulan vakum dari kelas offline, akhirnya program public speaking secara offline telah kembali untuk regional Yogyakarta. Tema menyusun speech dengan jumlah pertemuan intensive sebanyak 10x bertatap muka. Kedepannya kami akan memperluas jaringan kami ke beberapa kota-kota besar di Indonesia. Tunggu kehadiran kami yaa. Doakan proyek ini bisa berjalan dengan cepat.

Upcoming Class

Ada 2 Program Baru yang akan segera meluncur di Dialogika. Pertama, Academic Presentation. Sebuah program belajar Public Speaking untuk presentasi dengan bahasa inggris. Program ini akan hadir dalam kelas ekslusif maksimal peserta 5 orang. Kedua, Master of Ceremony. Program ini akan mengajarkan kepada teman-teman bagaimana cara menjadi seorang Professional MC.
